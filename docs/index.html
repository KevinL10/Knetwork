‹ HOW IT WORKS

Wikipedia, Khan Academy, Quizlet, OpenUniversity, and more - offline.

Existing ed-tech SaaSes focus on existing privilege, serving students who already have internet access and home devices. A Knetwork box is easy to configure and powerful even offline. Download a disk image packaged with tonnes of educational content, an offline web browser, and our local Knetwork app that allows students to access ML-generated questions and allows teachers to track student progress.

FEATURES ›

A Knetwork Knnects students.

Plug in a topic to search thousands of pages downloaded to a Knetwork box. Students can follow their own curiosity with unlimited, dynamically-generated questions at their own page, or tackle topics set by a supervisor. Teachers can view student progress and check in on each student's work. And of course, free, open source, and offline.

INFINITE QUESTIONS

Natural-language processing tools generate an infinite suite of problems for students.

SO, SO CHEAP

A single Raspberry Pi Zero, at $15, is sufficient, and additional storage can be added on via cheap microSD cards.

OPEN SOURCE

Fully free for all and open source - no recurring subscriptions or locked-in features.

LOW POWER

A Knetwork box can be solar powered, especially useful in communities without or with limited access to electricity.

Equipment Cost

Base Tier

Completely free and open source.

  • 1 Raspberry Pi Zero 2 W ($15)
  • 128 GB microSD card ($25)
  • 5V Solar Panels ($18)

< $60

More storage

Add on more storage with more micro SD cards.

  • 128 GB - enough, and more, for all of Wikipedia ($25)
  • 256 GB - enough to add on all of Khan Academy and more ($30)
  • 512 GB - enough to add on several full-length MOOCs and videos and more ($60)

More speed

Higher-powered Raspberry Pis will run Knetwork faster.

  • Raspberry Pi Zero 2 W - enough for Knetwork ($15)
  • Raspberry Pi 3 Model B+ - quad-core processor ($50)
  • Raspberry Pi 4 Model B - quad-core, and up to 8GB of RAM ($35)

Frequently Asked Questions

The Internet in a Box concept was what we were initially inspired by! However, we saw a potential for increased randomization and guidance for students, as well as an opportunity to empower teachers to check in on student learning. We also wanted to experiment with ML and NLP on low-powered devices, which IiaB models don't utilize.

Teachers in remote, offline communities aren't able to access the learning management systems that many North American universities or schools use. As well, full-fledged SaaSes would likely be out of budget, especially given purchasing power parity and subscription-based models. We wanted to target these teachers to give them the power of a supervisory tool, without the cost of American-targeted systems.

This hackathon, we weren't able to tackle everything we wanted to in 24 hours. However, we'd like to consider:
  • Compressing content even more, so we can pack more downloaded information in
  • Using ML techniques to explain concepts in the culture of a specific community
  • Translating topics and the entire app into native languages
  • Translating downloaded video in real-time into a native language
  • Making a bundled installer so users don't have to be tech-savvy

About the project

Knetwork is our team's submission to HackTheChange 2022. We saw an underserved niche in serving communities that lacked existing Internet infrastructure or access to individual student devices. We aimed to combine cheap hardware, lightweight software, and advanced ML techniques to create an infinite repository of knowledge for students to leverage. Hope you found this project as exciting as we did!
